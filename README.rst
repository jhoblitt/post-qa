#######
post-qa
#######

Upload QA datasets from tools like `validate_drp <https://github.com/lsst/validate_drp>`_ to the SQuaSH database and web app.
``post-qa`` is meant to run in a CI workflow, like LSST Data Management's Jenkins CI.
